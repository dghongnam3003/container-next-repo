# Next.js Project

A [Next.js](https://nextjs.org) application created with `create-next-app`.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

Edit `app/page.tsx` to start developing.
